# Sistema Bancario
Este projeto é um simulador de operações bancárias desenvolvido exclusivamente em Python, focado na aplicação de lógica de programação estruturada.
Esse projeto tem sistemas basicos de validação, tanto de dados quanto de transções.
É um projeto que ainda esta em manutenção e tem muita coisa a ser melhorada.

# Objetivo
O objetivo é consolidar conhecimentos em controle de fluxo, estruturas de dados e tratamento de exceções, sem dependência de frameworks externos ou bancos de dados complexos
